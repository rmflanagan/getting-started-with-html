The Basics of the Web & HTML

Basics of the Web

The world wide web is comprised of a large collection of HTML documents. These documents are connected to each other by hyper links. HTML documents may contain plain text, html, images, videos and/or music.

The main components of the web are you, the user, your computer with its web browser, the internet and servers. The user goes to a web page (e.g. http://www.google.com) and your browser sends an http request via the internet to servers; these servers respond to these requests by locating the appropriate page(s) and displaying it back on your web browser. This video contains a visual representation of these web component relationships.

HTML

HTML is an abbreviation of hypertext markup language. HTML is comprised of text content, or what the user sees, markup, or how the user sees it, references to other documents, such as images or videos, and finally links to other pages. This video gives an HTML overview.

HMTL Tags/Elements

HTML markup is written as elements that describe how contents are displayed in a document. To write these elements, we use HTML tags. This video shows the structure of tags. Examples of basic tags are those used for displaying bold text (b for bold) and italic text (em for emphasis).

A "Smart" Aside about "Stupid" Computers

Computers are "stupid" in the sense that they can only literally translate programming code. They cannot interpret what a programmer intended, and are therefore unable to correct mistakes programmers make. Even small errors in code can lead to large problems in the rendered results.

HTML Attributes

Some HTML tags have attributes, which use an additional value in the opening tag. An example is the anchor tag, as described in this video, that uses the attribute "href" to turn text into a link to another page. A further example is the image tag, that contains the attribute value "scr" to display an image file.

Another Aside about Remembering

It's important to note that even experienced programmers don't remember how to code everything, as there is simply too much to remember! If you forget how to do something, a simple online search will typically give you the answer you need.

Inline/Block Elements

HTML elements are classified as either inline or block. Inline elements are just text, whereas block elements create an "invisible box" around text. This video uses the break and paragraph tags to describe the distinction between inline and block tags.

Container Elements

Container elements simply "contain" whatever is found within them. The example given in this video is "span", an inline element, and "div", a block element.

HTML Document Structure

HTML documents need doctype, html, head and body tags. Doctype has been simplified since the use of html5. The head section may contain things such as the document meta-data, CSS styling elements and the title of the page. The body section contains the actual text of the page.
